# Globally

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Built With
* [Vue.js](https://vuejs.org/)
* [TailwindCSS](https://tailwindcss.com/)
* [Covid19api](https://covid19api.com/)
